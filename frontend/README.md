# Wine Cellar Database & Showcase

Este proyecto es una base de datos curada de bodegas emblemáticas de Mendoza, diseñada con una estética elegante y minimalista inspirada en la cultura vitivinícola.

## 🍷 Características
* **Catálogo de Bodegas:** Información técnica detallada sobre las bodegas más icónicas de Argentina.
* **Soporte Bilingüe:** Descripciones disponibles tanto en español como en inglés.
* **Diseño Premium:** Estilos CSS personalizados con tipografía *Playfair Display* y paleta de colores inspirada en vinos tintos (color #722f37).

## 🛠 Tecnologías Utilizadas
* **HTML5**
* **CSS3** (con fuentes de Google Fonts)
